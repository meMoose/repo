Description:

This command line allow you to upload deb packages to cydia repo with a single click, it can be usefull for developers who dealing with upload/reupload packages alot, also it has feature to modifying control file of deb packages before uploading, such Author, Depends, Version etc.
usage : debup [github repo url] [deb directory path] [opt]
